Detailed Description

This interface provides a standardized way to interact with WebSocket connections. It defines four pure virtual functions: Open(), Close(), Send(message), and OnReceive(message). Any class that implements this interface must provide its own implementation of these functions. The virtual destructor ~IWebSocket() is also declared to ensure proper release of resources when the interface is destroyed.
